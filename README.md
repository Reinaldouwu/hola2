Licensia.py
## Base Y Caracteristicas Generales

> Este es un sistema de licencia basico done hace preguntas basicas, (por ejemplo:Tu edad, si aprobaste el examen de conducir, [Teorico o practico],y si tengo una licencia valida).
esto se basa creando una lista con todos los requisitos y mediante comandos Basicos como: (if, else, elif, while.) se determina si eres capaz de obtener una licencia de conducir, tambien puede corregir errores por ejemplo si utilizo letras donde van numeros se muestra un mensaje de advertencia  te da la opcion de corregirlo y ingresar un numero valido sin reiniciar el programa.

> al final los elementos de la lista se chequean mediante un if all(requisitos) print=yes else print=no.

>  Estos requisitos se apruban mediante de la transformacion de las respuestas de las preguntas que se convierten en variables de tipo booleano( exceptuando la variable EDAD que tiene un bug que pasa directamente de valor numerico a boleano chocando con unas lineas de codigo mas abajo[!NOTE: SI ES BOOLEANO PERO LO IDEAL ES CREAR OTRA VARIABLE])

## advertencias y limitaciones del proyecto: 

! al ser de mis primeros codigos y ser un proyecto muy Basico, se toman en cuenta que las siguientes variables este establecidas en VERDADERO por defecto; identificacion= TRUE(la identificacion esta vigente y en regla) y Examen_P(examen practico que tambien se toma TRUE como valor por defecto lo que quiere decir que fue aprobado)
