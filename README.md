Licensia.py
-Este es un sitema de licencia basico done hace preguntas basicas, (por ejemplo:Tu edad, si aprobaste el examen de conducir, [Teorico o practico],y si tengo una licencia valida).
esto se basa creando una libreria con todos los requisitos y mediante comandos Basicos como: (if, else, elif, while.) se determina si eres capaz de obtener una liscencia de conducir, tambien puede corregir errores por ejemplo si utilizo letras donde van numeros se muestra un mensaje de advertencia  te da la opcion de corregirlo y ingresar un numero valido sin reiniciar el proceso desde 0.
al final los elementos de la lista se chequean mediante un if all(requisitos) print=yes else print=no

! al ser de mis primeros copdigos y ser un prooyecto muy Basico, se toman en cuenta que las siguientes variables este establecidas en VERDADERO por defecto; identificacion= TRUE(la identificacion esta vigente y en regla) y Examen_P(examen practico que tambien se toma TRUE como valor por defecto lo que quiere decir que fue aprobado)
